# Manual del estudiante

1. Registra una cuenta con el rol **Estudiante**.
2. Completa tu perfil con escuela profesional, teléfono y enlace a tu CV.
3. Ingresa a **Convocatorias** y usa los filtros para encontrar oportunidades.
4. Selecciona **Postular** en una convocatoria activa.
5. Revisa **Mis postulaciones** para conocer el estado de cada solicitud.

No es posible postular dos veces a la misma convocatoria ni postular a ofertas cerradas.
