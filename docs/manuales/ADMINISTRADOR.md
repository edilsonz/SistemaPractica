# Manual del administrador

1. Inicia sesión con una cuenta administrativa creada de forma controlada.
2. Usa el **Dashboard** para revisar métricas generales.
3. En **Empresas pendientes**, verifica los datos institucionales y aprueba solo organizaciones válidas.
4. Supervisa convocatorias y postulaciones desde los módulos correspondientes.
5. Revisa el historial de cambios y aplica la política de auditoría establecida para el despliegue institucional.
